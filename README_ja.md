# TraceMark

[English](README.md) | [简体中文](README_zh.md) | [繁體中文](README_zh-Hant.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

TraceMark（トレースマーク）は、macOS（macOS 13以降）向けに構築された効率的で軽量なシステムレベルのスクリーンショット＆アノテーションツールです。スクリーンショットに基づいた明確で整理されたアノテーション（注釈）をすばやく作成するのに役立ちます。

## ✨ 主な機能

- **おすすめ機能**:
  - 💬 **連番テキスト (吹き出し)**: 自動的に増加する番号と、自由にドラッグ可能な吹き出しテキストボックス。手順の指示を明確かつ非常に読みやすくし、デザインレビューやフィードバックに最適です。
  - **履歴ダッシュボード**: 過去のスクリーンショットとアノテーションの再編集をサポートします。細部の修正のために再度スクリーンショットを撮る必要はありません。過去のスクリーンショットをデスクトップにピン留めして簡単に参照することもできます。
  - **OCR (光学文字認識)**: スクリーンショットからテキストを瞬時に抽出します。
  - **翻訳**: ワンクリック翻訳（macOS 14.4以降のみ）。
  - 多言語対応 (i18n): 英語、中国語、日本語、韓国語のインターフェースをネイティブサポートしています。

- **豊富な注釈ツール**:
  - 🖌️ **ブラシとハイライト**: フリーハンドの描画とテキストのハイライト。
  - 📏 **図形**: 長方形、楕円、矢印などの標準的な図形。
  - 💧 **モザイクとぼかし**: プライバシーや機密データを保護するための必須ツール。
  - **グローバルショートカット**: カスタマイズ可能なグローバルショートカットで、画面を瞬時にフリーズします。

## 📥 インストールガイド

> ⚠️ **注意**: 現在のバージョンには Apple Developer 署名がないため、macOS のデフォルトの Gatekeeper メカニズムによって、未確認の開発者からのアプリの実行がブロックされます。アプリを承認して実行するには、以下の手順に従ってください。

### ダウンロードとインストール
1. このリポジトリの [Releases](#) ページに移動し、最新の `TraceMark.dmg` をダウンロードします。
2. DMG ファイルを開き、`TraceMark.app` を **アプリケーション (Applications)** フォルダにドラッグします。

### Gatekeeper のバイパス (インストール時必読)

「TraceMark は壊れているため開けません」または「開発元が未確認」のプロンプトが表示された場合、最も効果的な推奨方法は「ターミナル」を使用してアプリケーションの隔離属性を削除することです。

**手順:**
1. **ターミナル (Terminal)** アプリを開きます（Spotlight 検索 `Cmd + Space` で「ターミナル」と入力して開くことができます）。
2. 次のコマンドをコピーしてターミナルウィンドウに貼り付けます:
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/TraceMark.app
   ```
3. **Enter** キーを押します。
4. ターミナルで `Password:` が求められます。Mac のログインパスワードを入力してください（※入力中、画面には文字や星印は表示されませんが、これは通常のセキュリティメカニズムです）。
5. もう一度 **Enter** キーを押します。
6. 完了です！これでアプリケーションフォルダから通常通り TraceMark をダブルクリックして開くことができます。

### 権限の許可ガイド (初回実行時)

TraceMark が正常に機能するために、システムは初回起動時に必要な権限を取得する必要があります。TraceMark を初めて実行すると、システムから自動的に承認のプロンプトが表示されます。

1. **画面収録 (必須)**: 画面の画像をキャプチャするために使用します。プロンプト内の「システム設定を開く」をクリックし、TraceMark を見つけてスイッチをオンにします。
2. **アクセシビリティ (推奨)**: グローバルショートカットキーをリッスンするために使用します。プロンプト内の「システム設定を開く」をクリックし、TraceMark を見つけてスイッチをオンにします。

> 誤ってプロンプトを拒否してしまった場合は、「システム設定」 > 「プライバシーとセキュリティ」からいつでも手動で有効にすることができます。

### ショートカットキーのカスタマイズ

TraceMark のデフォルトのスクリーンショットショートカットは `Option + A` です。これがシステムの他のショートカットと競合する場合は、以下の手順でカスタマイズできます。

1. TraceMark を開いた後、メニューバー（画面右上）の TraceMark アイコンをクリックします。
2. **「環境設定...」 (Preferences...)** を選択します。
3. ポップアップした設定ウィンドウで、「スクリーンショットショートカット」オプションを探します。
4. 現在のショートカットボタンをクリックし、設定したい新しいキーの組み合わせ（例：`Control + Cmd + A` や `Cmd + Shift + A`）を直接キーボードで押します。
5. 設定は即座に反映されます。

## 🛠 開発とビルド

ソースから TraceMark をビルドしたい場合は、macOS 13 以降および Xcode 14 以降がインストールされていることを確認してください。

1. リポジトリのクローン:
   ```bash
   git clone https://github.com/your-username/TraceMark.git
   cd TraceMark
   ```
2. アプリのビルドとパッケージ化:
   ```bash
   sh scripts/build-app.sh && sh scripts/build-dmg.sh
   ```
3. ビルドされた製品は `build/TraceMark.dmg` に出力されます。
