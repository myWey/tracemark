# TraceMark

[English](README.md) | [简体中文](README_zh.md) | [繁體中文](README_zh-Hant.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

TraceMark는 macOS(macOS 13 이상)를 위해 제작된 효율적이고 가벼운 시스템 수준의 화면 캡처 및 주석 도구입니다. 스크린샷을 기반으로 명확하고 체계적인 주석 설명을 빠르게 작성할 수 있도록 도와줍니다.

## ✨ 핵심 기능

- **추천 기능**:
  - 💬 **번호 매기기 텍스트 (콜아웃)**: 자동으로 증가하는 번호와 자유롭게 드래그할 수 있는 콜아웃 텍스트 상자를 통해 단계별 설명을 명확하고 읽기 쉽게 만듭니다. 디자인 리뷰 및 피드백에 매우 유용합니다.
  - **기록 대시보드**: 이전 스크린샷 및 주석의 재편집을 지원하여 세부 사항을 수정하기 위해 스크린샷을 다시 찍을 필요가 없습니다. 이전 스크린샷을 바탕화면에 고정하여 쉽게 참고할 수도 있습니다.
  - **OCR (광학 문자 인식)**: 스크린샷에서 텍스트를 즉시 추출합니다.
  - **번역**: 원클릭 번역 (macOS 14.4 이상 전용).
  - 다국어 지원 (i18n): 한국어, 영어, 중국어, 일본어 인터페이스를 기본으로 지원합니다.

- **풍부한 주석 도구**:
  - 🖌️ **브러시 및 하이라이트**: 자유로운 그리기 및 텍스트 강조.
  - 📏 **도형**: 직사각형, 타원, 화살표 등 표준화된 도형.
  - 💧 **모자이크 및 블러**: 개인 정보와 민감한 데이터를 보호하기 위한 필수 도구.
  - **글로벌 단축키**: 사용자 정의 가능한 단축키로 화면을 즉시 캡처합니다.

## 📥 설치 가이드

> ⚠️ **주의**: 현재 버전은 아직 Apple Developer 서명이 없기 때문에 macOS의 기본 Gatekeeper 메커니즘이 확인되지 않은 개발자의 앱 실행을 차단합니다. 앱을 승인하고 실행하려면 아래 단계를 따르세요.

### 다운로드 및 설치
1. 이 저장소의 [Releases](#) 페이지로 이동하여 최신 `TraceMark.dmg`를 다운로드합니다.
2. DMG 파일을 열고 `TraceMark.app`을 **응용 프로그램 (Applications)** 폴더로 드래그합니다.

### Gatekeeper 우회 (설치 시 필수 참고)

"TraceMark가 손상되어 열 수 없습니다" 또는 "확인되지 않은 개발자" 프롬프트가 표시되는 경우 가장 효과적인 방법은 "터미널"을 사용하여 앱의 격리 속성을 제거하는 것입니다.

**단계별 지침:**
1. **터미널 (Terminal)** 앱을 엽니다 (Spotlight 검색 `Cmd + Space`를 사용하여 "터미널"을 입력하여 찾을 수 있습니다).
2. 터미널 창에 다음 명령을 복사하여 붙여넣습니다:
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/TraceMark.app
   ```
3. **Enter** 키를 누릅니다.
4. 터미널에 `Password:` 프롬프트가 표시됩니다. Mac 로그인 비밀번호를 입력합니다 (※입력 시 화면에 문자나 별표가 표시되지 않지만 이는 정상적인 보안 메커니즘입니다).
5. 다시 한 번 **Enter** 키를 누릅니다.
6. 완료되었습니다! 이제 응용 프로그램 폴더에서 TraceMark를 정상적으로 두 번 클릭하여 열 수 있습니다.

### 권한 설정 가이드 (최초 실행 시)

TraceMark가 정상적으로 작동하려면 시스템이 최초 실행 시 필요한 권한을 획득해야 합니다. TraceMark를 처음 실행하면 시스템에서 자동으로 권한 부여 프롬프트를 표시합니다:

1. **화면 기록 (필수)**: 화면 이미지를 캡처하는 데 사용됩니다. 프롬프트에서 "시스템 설정 열기"를 클릭하고 TraceMark를 찾아 스위치를 켭니다.
2. **손쉬운 사용 (권장)**: 글로벌 단축키를 감지하는 데 사용됩니다. 프롬프트에서 "시스템 설정 열기"를 클릭하고 TraceMark를 찾아 스위치를 켭니다.

> 실수로 프롬프트를 거부한 경우 언제든지 「시스템 설정」 > 「개인정보 보호 및 보안」으로 이동하여 수동으로 켤 수 있습니다.

### 단축키 사용자 정의

TraceMark의 기본 스크린샷 단축키는 `Option + A`입니다. 이 단축키가 시스템의 다른 단축키와 충돌하는 경우 다음 단계에 따라 사용자 정의할 수 있습니다:

1. TraceMark를 연 후 메뉴 막대(화면 오른쪽 상단)에서 TraceMark 아이콘을 클릭합니다.
2. **「환경설정...」 (Preferences...)**을 선택합니다.
3. 팝업된 설정 창에서 "스크린샷 단축키" 옵션을 찾습니다.
4. 현재 단축키 버튼을 클릭한 다음 원하는 새로운 키 조합(예: `Control + Cmd + A` 또는 `Cmd + Shift + A`)을 키보드에서 직접 누릅니다.
5. 설정이 완료되면 즉시 적용됩니다.

## 🛠 개발 및 빌드

소스에서 TraceMark를 빌드하려면 macOS 13 이상 및 Xcode 14 이상이 설치되어 있는지 확인하세요.

1. 저장소 클론:
   ```bash
   git clone https://github.com/your-username/TraceMark.git
   cd TraceMark
   ```
2. 앱 빌드 및 패키징:
   ```bash
   sh scripts/build-app.sh && sh scripts/build-dmg.sh
   ```
3. 빌드된 제품은 `build/TraceMark.dmg`에 출력됩니다.
