# TraceMark

[English](README.md) | [简体中文](README_zh.md) | [繁體中文](README_zh-Hant.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

TraceMark 是一款為 macOS（macOS 13 及以上）打造的高效、輕量的系統級截圖與標註工具，能夠幫助你快速地基於截圖做清晰、有序的標註說明。

## ✨ 核心特性

### 1. 更優雅的標註體驗

- **💬 序號 & 帶序號文字（引線標註）**：自動遞增序號，配合可自由拖曳的引線文字方塊，讓步驟說明排版清晰、閱讀友好，設計走查與產品反饋一目瞭然。
- **📝 矩形框文本**：透明填充、僅保留邊框的矩形文本標註，適合框選說明特定區域，視覺更簡潔專業。
- **🪟 視窗自動吸附**：截圖時自動識別視窗邊界，過濾多餘邊緣背景，輸出更乾淨。
- **🎨 精緻配色**：基於 macOS 系統語義顏色與統一紫色 AI 主題，介面與輸出都保持專業美感。

### 2. 智慧化工具

- **🤖 給 AI 定位**：在截圖上標記關鍵區域，一鍵導出原圖 + 歸一化座標 `[xmin, ymin, xmax, ymax]` 的 prompt，讓 AI 精準定位、指哪改哪。
- **🔍 OCR 文字辨識**：一鍵提取截圖中的文字內容，支援複製純文字或含像素座標的完整資訊。
- **🌐 一鍵翻譯**：快速翻譯截圖中的文字內容（macOS 14.4 及以上）。

### 3. 更高效的截圖管理

- **🗂️ 歷史記錄歸檔**：截圖與標註自動歸檔，支援二次編輯，不必為細節修改重新截圖。
- **📌 貼圖釘住置頂**：將截圖以置頂浮窗形式釘在桌面，支援多張、自由縮放，參考對照更方便。

### 通用能力

- **全域快捷鍵喚起**：可自定義快捷鍵，一鍵定格螢幕，支援多螢幕同時截圖。
- **多語言支援 (i18n)**：原生支援中文、英文、日語、韓語介面。

## 📥 安裝指南

> ⚠️ **注意**：由於當前版本暫未申請 Apple Developer 簽名，macOS 預設的 Gatekeeper 機制會阻止未知開發者的應用程式執行。請按照以下步驟進行信任授權。

### 下載與安裝
1. 前往本倉庫的 [Releases](#) 頁面下載最新的 `TraceMark.dmg`。
2. 打開 DMG 檔案並將 `TraceMark.app` 拖入到您的 **應用程式 (Applications)** 資料夾中。

### 繞過 Gatekeeper (安裝必讀)

如果您執行應用程式時遇到「TraceMark 已損壞，無法打開」或「未知的開發者」的提示，最徹底有效的推薦方法是使用終端機（Terminal）解除應用程式的隔離屬性。

**具體操作步驟：**
1. 打開 **終端機** 應用程式（可以透過快捷鍵 `Cmd + 空白鍵` 呼出 Spotlight 搜尋，輸入「終端機」或「Terminal」並按下 Return 打開）。
2. 將以下指令**複製並貼上**到終端機視窗中：
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/TraceMark.app
   ```
3. 按下 **Return** 鍵。
4. 此時終端機會提示輸入 `Password:`。請直接盲打輸入您的 Mac 開機/鎖定密碼（注意：輸入密碼時螢幕上不會顯示任何字元或星號，這是正常的安全機制）。
5. 輸入完畢後，再次按下 **Return** 鍵。
6. 完成！現在您可以去應用程式資料夾，直接點兩下正常打開 TraceMark 了。

### 權限開啟指南 (首次執行)

為了確保 TraceMark 能夠正常運作，系統需要在首次啟動時獲取必要的權限。當您第一次運行 TraceMark 時，系統會自動彈出授權提示視窗：

1. **螢幕錄製權限 (必須)**：用於擷取螢幕圖像。請在彈出的系統提示框中點擊「打開系統設定」，找到 TraceMark 並將其開關打開。
2. **輔助使用權限 (推薦)**：用於全域快捷鍵喚醒。請在彈出的系統提示框中點擊「打開系統設定」，找到 TraceMark 並將其開關打開。

> 如果您不小心拒絕了，可以隨時手動前往「系統設定」 > 「隱私權與安全性」中進行手動開啟。

### 自定義快捷鍵設定

TraceMark 的預設截圖快捷鍵為 `Option + A`。如果您發現與系統其他快捷鍵衝突，您可以按以下步驟自定義：

1. 打開 TraceMark 後，在狀態列（螢幕右上角）點擊 TraceMark 圖標。
2. 選擇 **「偏好設定...」 (Preferences...)**。
3. 在彈出的設定視窗中，找到「截圖快捷鍵」選項。
4. 點擊目前的快捷鍵按鈕，然後直接在鍵盤上按下您想要設定的新組合鍵（例如 `Control + Cmd + A` 或 `Cmd + Shift + A`）。
5. 設定完成後立即生效。

## 🛠 開發與建置

如果您希望從原始碼建置 TraceMark，請確保安裝了 macOS 13+ 和 Xcode 14+。

1. 複製倉庫：
   ```bash
   git clone https://github.com/your-username/TraceMark.git
   cd TraceMark
   ```
2. 建置並打包應用程式：
   ```bash
   sh scripts/build-app.sh && sh scripts/build-dmg.sh
   ```
3. 建置產物將會出現在 `build/TraceMark.dmg`。
