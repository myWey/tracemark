# 产品哲学（Philosophy）

> 做权衡时会引用 `P1`、`A1` 这种段落 ID。

## 定位（一句话）

一个 Mac 平台上的轻量、简单、高效、易用的本地化截图工具，作为 CleanShot X 的高价值杠杆功能平替，采用一次性合理收费（或对大众更友好的模式），专注于提供最核心、最稳定、高频好用的截图与标注体验。

## 核心用户痛点（带具体场景）

1. **核心高阶功能付费门槛高** —— 场景：Mac 用户在日常办公、写文档或沟通中，需要像滚动截图、截图贴图（Pin）、精美窗口阴影等 CleanShot X 级别的精致体验。当下的 workaround：使用系统自带截图（缺少贴图和高级标注）或者购买高价的 CleanShot X 授权。代价：高昂的软件授权成本。
2. **免费工具不够独立/轻量，存在隐私隐患** —— 场景：用户不想为了截图而启动微信、QQ 等大型社交软件。当下的 workaround：启动微信/QQ 并挂载后台，使用其快捷键截图。代价：内存消耗大，且敏感工作截图可能面临隐私泄露风险。
3. **截图分享过程中的信息脱敏繁琐** —— 场景：开发者或产品经理向外部发送截图，需要遮蔽密钥、手机号或头像等敏感信息。当下的 workaround：使用马赛克或画笔工具手动模糊处理。代价：操作繁琐，易遗漏，效率低下。

## 受众

- **主受众（必须爱上）**：Mac 深度用户、开发者、产品经理、设计师、写文档/教程的技术博主。他们截图频次极高，非常看重截图的视觉品质（如精美背景/阴影）、标注效率和贴图比对功能。
- **次受众（顺带服务）**：普通办公 Mac 用户。他们只需要偶尔进行精致截图、滚动截取长网页或长聊天记录，对于操作流畅度有一定追求。
- **明确不服务的**：需要复杂云端团队协作的用户（不做大型团队云端管理）；Windows/Linux 用户（专注于 macOS 极致原生体验）；需要专业级图像/画板编辑的用户（不做 Figma / Photoshop 平替，只做轻量截图与标注）。

## 设计原则（按优先级，可判断）

- **P1. 本地与隐私优先** —— 当“云端协同便利性”与“本地运行的极速、隐私安全”冲突时，倾向于**本地与隐私**。我们绝不强制联网、不上传用户截图，所有数据默认仅留在用户本地。
- **P2. 高价值杠杆与极简** —— 当“功能大而全”与“轻量易用”冲突时，倾向于**轻量易用**。专注于 20% 能满足 80% 高频场景的核心功能（如高清截图、贴图 Pin、滚动长截图、高阶阴影与智能脱敏），不堆砌专业级图形画板。
- **P3. 高效与零摩擦交互** —— 当“配置参数的多样性”与“一键即达的直观性”冲突时，倾向于**一键即达**。软件在后台应该“隐形”不占资源，截图交互路径做到最短（拖拽即分享、智能吸附脱敏）。
- **P4. 精致与高品质视觉** —— 当“快速粗糙的开发”与“精致视觉细节（如窗口圆角、软阴影、背景色卡）”冲突时，倾向于**精致视觉细节**。产出的截图必须有高颜值，让用户在分享时体现专业感。

## 反模式（永远不做）

- **A1. 强制订阅与云服务捆绑** —— 绝不强迫用户购买云空间以使用基础的本地截图、标注或贴图功能。提供纯本地买断的良心体验。
- **A2. 后台静默运行高开销** —— 绝不在后台常驻庞大内存（如 Electron 空载占用 200MB+），必须做到启动秒开、后台常驻低开销。
- **A3. 强打扰与推送** —— 没有广告、没有红点打扰、没有弹窗推送，做一个安静且优雅的生产力工具。

## 路线图与实施规划（Specs Roadmap）

本项目采用分段 Spec 实施方案，以逐步对齐 CleanShot X 的高价值杠杆功能：

### Spec-1: 区域截图与悬浮分发（MVP 基础）
- **核心目的**：构建 macOS 全局快捷键与透明遮罩窗口捕获流程，打通最频繁的截图与文件处理闭环。
- **功能点**：
  1. 菜单栏图标与全局快捷键（默认 `Cmd + Shift + X`）唤醒。
  2. 覆盖多屏的透明区域选择器（Overlay Canvas），拖拽选区像素捕获。
  3. 右下角悬浮缩略图，展示截图内容，支持拖拽导出、快捷复制/保存/关闭动作。

### Spec-2: 精致标注与智能脱敏（Annotation Core）
- **核心目的**：提供比系统自带截图更精美、快捷的图像微调与敏感信息遮蔽功能。
- **功能点**：
  1. 矢量标注层：箭头、直线、椭圆、序号、高亮框、文本。
  2. 智能脱敏（亮点）：一键识别框选范围内的敏感信息（如手机号、代码 Token、头像）并自动高斯模糊/马赛克。
  3. 精美画布美化：为截图一键增加视网膜圆角、多种渐变/单色底色卡，以及平滑的软阴影背景。

### Spec-3: 置顶贴图与长图滚动（Power Tools）
- **核心目的**：支持高频的临时数据比对与滚动长文档截取。
- **功能点**：
  1. 贴图管理器（Pin Window）：将截图以无边框窗口形态钉在桌面上置顶，支持滚轮调节透明度、双击最小化。
  2. 滚动截图（Scrolling Capture）：自动捕获网页或滚动视图的内容并拼接为长图。
