# TraceMark

[English](README.md) | [简体中文](README_zh.md) | [繁體中文](README_zh-Hant.md) | [日本語](README_ja.md) | [한국어](README_ko.md)

TraceMark 是一款为 macOS 打造的高效、轻量的系统级截图与标注工具。它融合了现代设计美学与强大的标注能力，能够帮助您快速捕获灵感并进行专业的图形化说明。

## ✨ 核心特性

- **全局快捷键唤起**：一键定格屏幕，支持多屏幕同时截图。
- **丰富的标注工具**：
  - 💬 **序号 & 带序号文本 (引线标注)**：自动递增的序号，外加可自由拖拽的引线文本框，让标注步骤清晰、可读性更强，是设计走查与产品反馈的利器。
  - 🖌️ **画笔与高亮**：自由绘制与文本高亮。
  - 📏 **形状**：矩形、椭圆、箭头等标准化形状。
  - 💧 **马赛克与模糊**：保护隐私与敏感数据的必备工具。
  - 📌 **贴图钉住**：将截图内容固定在屏幕顶层，方便参考。
- **智能化工作流**：
  - **历史记录**：支持对历史截图与标注的再次修改编辑（注：目前版本暂未支持历史记录搜索功能）。
  - **OCR 文本识别**：一键提取截图中的文字内容。
  - 多语言支持 (i18n)：原生支持中文、英文、日语、韩语等多语言界面。

## 📥 安装指南

> ⚠️ **注意**：由于当前版本暂未申请 Apple Developer 签名，macOS 默认的 Gatekeeper 机制会阻止未知开发者的应用运行。请按照以下步骤进行信任授权。

### 下载与安装
1. 前往本仓库的 [Releases](#) 页面下载最新的 `TraceMark.app.zip`。
2. 解压并将 `TraceMark.app` 拖入到您的 **应用程序 (Applications)** 文件夹中。

### 绕过 Gatekeeper (小白必读推荐方法)

如果您运行应用时遇到“TraceMark 已损坏，无法打开”或“未知开发者”的提示，最彻底有效的推荐方法是使用终端（Terminal）解除应用的隔离属性。

**具体操作步骤：**
1. 打开 **终端** 应用（可以通过快捷键 `Cmd + 空格` 呼出聚焦搜索，输入“终端”或“Terminal”并回车打开）。
2. 将以下命令**复制并粘贴**到终端窗口中：
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/TraceMark.app
   ```
3. 按下 **回车键 (Enter)**。
4. 此时终端会提示输入 `Password:`。请直接盲打输入您的 Mac 开机/锁屏密码（注意：输入密码时屏幕上不会显示任何字符或星号，这是正常的安全机制）。
5. 输入完毕后，再次按下 **回车键 (Enter)**。
6. 完成！现在您可以去应用程序文件夹，直接双击正常打开 TraceMark 了。

### 权限开启指南 (首次运行)

为了确保 TraceMark 能够正常运作，系统需要在首次启动时获取必要的权限。当您第一次运行 TraceMark 时，系统会自动弹出授权提示窗口：

1. **屏幕录制权限 (必须)**：用于截取屏幕图像。请在弹出的系统提示框中点击“打开系统设置”，找到 TraceMark 并将其开关打开。
2. **辅助功能权限 (推荐)**：用于全局快捷键唤醒。请在弹出的系统提示框中点击“打开系统设置”，找到 TraceMark 并将其开关打开。

> 如果您不小心拒绝了，可以随时手动前往「系统偏好设置」 > 「隐私与安全性」中进行手动开启。

### 自定义快捷键设置

TraceMark 的默认截图快捷键为 `Option + A`。如果您发现与系统其他快捷键冲突，您可以按以下步骤自定义：

1. 打开 TraceMark 后，在状态栏（屏幕右上角）点击 TraceMark 图标。
2. 选择 **「偏好设置...」 (Preferences...)**。
3. 在弹出的设置窗口中，找到「截图快捷键」选项。
4. 点击当前的快捷键按钮，然后直接在键盘上按下您想要设置的新组合键（例如 `Control + Cmd + A` 或 `Cmd + Shift + A`）。
5. 设置完成后立即生效。

## 🛠 开发与构建

如果您希望从源码构建 TraceMark，请确保安装了 macOS 13+ 和 Xcode 14+。

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-username/TraceMark.git
   cd TraceMark
   ```
2. 构建并打包应用：
   ```bash
   sh scripts/build-app.sh
   ```
3. 构建产物将会出现在 `build/TraceMark.app`。
